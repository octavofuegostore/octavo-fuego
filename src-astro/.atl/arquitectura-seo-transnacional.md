## 2. Mapa de Dominios y Responsabilidad SEO

```
octavofuego.com/es/              → Detal Colombia (COP) — Español
octavofuego.com/es/mayoristas/   → B2B Colombia — Español (noindex portal)
octavofuego.com/en/              → Detal + Wholesale Internacional (USD) — Inglés
octavofogo.com.br/               → Detal Brasil (BRL) — Portugués
octavofogo.com.br/atacado/       → B2B Brasil — Portugués (noindex portal)
```

### Regla de oro: un dominio, una audiencia, un idioma

Nunca mezclar contenido de detal con contenido mayorista en la misma URL indexada. Nunca mezclar idiomas en la misma ruta. Cada URL le habla a una sola persona con una sola intención.

## 7. Etiquetas `hreflang` Cruzadas

Estas etiquetas van en el `<head>` de todas las páginas indexadas. Le dicen a Google que los tres hubs son la misma marca para audiencias distintas.

```html
<!-- Ejemplo: página de producto Tisunú -->
<link rel="alternate" hreflang="es-CO" href="https://octavofuego.com/es/tienda/rape/tisunu/" />
<link rel="alternate" hreflang="en"    href="https://octavofuego.com/en/shop/rape/tsunu/" />
<link rel="alternate" hreflang="pt-BR" href="https://octavofogo.com.br/loja/rape/tisunu/" />
<link rel="alternate" hreflang="x-default" href="https://octavofuego.com/en/shop/rape/tsunu/" />

<!-- Ejemplo: categoría rapé -->
<link rel="alternate" hreflang="es-CO" href="https://octavofuego.com/es/tienda/rape/" />
<link rel="alternate" hreflang="en"    href="https://octavofuego.com/en/shop/rape/" />
<link rel="alternate" hreflang="pt-BR" href="https://octavofogo.com.br/loja/rape/" />
<link rel="alternate" hreflang="x-default" href="https://octavofuego.com/en/shop/rape/" />

<!-- Ejemplo: categoría kuripe -->
<link rel="alternate" hreflang="es-CO" href="https://octavofuego.com/es/tienda/kuripe/" />
<link rel="alternate" hreflang="en"    href="https://octavofuego.com/en/shop/kuripe/" />
<link rel="alternate" hreflang="pt-BR" href="https://octavofogo.com.br/loja/kuripe/" />
<link rel="alternate" hreflang="x-default" href="https://octavofuego.com/en/shop/kuripe/" />
```

**Regla crítica:** Los slugs de producto en Medusa deben ser idénticos en las tres variantes para que el mapeo de `currentPath` en el Layout funcione sin errores. Acordar la convención antes de crear el primer producto.

|---------|--------|-----------|
| `/tienda/rape/tisunu/` | `/shop/rape/tsunu/` | `/loja/rape/tisunu/` |
| `/tienda/rape/pixuri/` | `/shop/rape/pixuri/` | `/loja/rape/pixuri/` |
| `/tienda/rape/parika/` | `/shop/rape/parika/` | `/loja/rape/parika/` |

## 9. Posicionamiento de Precio en Páginas de Producto

El precio por gramo debe ser el elemento más visible en la página de producto. La comparación implícita con la competencia ocurre en la mente del usuario sin necesidad de nombrarla.

**Jerarquía visual recomendada:**

```
$3.500 COP / gramo          ← NÚMERO MÁS GRANDE
$70.000 COP — 20 gramos     ← Precio total secundario
────────────────────────────
Opciones:
  [ 10g — $35.000 ]
  [ 20g — $70.000 ]  ← Best value
  [ 30g — $100.000 ]
```

**Copy de apoyo en cada producto:**
> "20 gramos por el precio que otros cobran por 10."

Esta línea no nombra a ningún competidor, es verificable y convierte porque el usuario que ha comparado precios ya lo sabe.
